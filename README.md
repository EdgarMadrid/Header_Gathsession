## ✨ About the Project

**Header Gathsession** is a frontend project focused on building a modern header-hero section based on a Figma design, following real-world development practices.

This project was developed using **Vite** as a build tool and applying modular architecture with **components-based structure**.

Key concepts applied:

✅ Component-based architecture  
✅ Semantic HTML structure  
✅ SCSS for scalable styling  
✅ JavaScript for interactivity and structure  
✅ BEM methodology for class naming  
✅ Responsive design implementation  
✅ Pixel-perfect design fidelity  
✅ SVG asset usage  
✅ Hover and interactive states  

---

## 🎯 Goal

Recreate a high-fidelity header-hero section from a Figma design while applying professional frontend practices used in real projects.

Focus areas:

- Clean architecture (components)
- Maintainable SCSS structure
- Reusable CSS with BEM
- Responsive UI behavior
- Proper semantic HTML
- Vite project setup

## 🛠️ Run Locally

Clone repository:

```bash
git clone https://github.com/EdgarMadrid/Header_Gathsession.git
```

Enter project:

```bash
cd Header_Gathsession
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm run dev
```

Build project:

```bash
pnpm run build
```
